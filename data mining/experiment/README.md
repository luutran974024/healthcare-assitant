# WikiHow Data Crawler

This Python script crawls data from WikiHow articles.

## Installation

Make sure you have Python installed. Then, install the required dependencies by running:

```cmd
pip install -r requirements.txt
```

## Usage

1. Before running the code, create a folder named `data` in the project directory.
2. Run the following command:

```python
python main.py
```


![Example](https://github.com/danghieuan/IVS_AI_Healthcare_intern/blob/dat_rp/mdattr/experiment/img/1.png)

## Output

The script will save the crawled data as JSON files in the `data` folder.
